Creating Cards:

To create additional cards to use in the game you have to provide adequate data to fill a list of keys. In order, these are the required fields which must (mostly) me provided.

Data:
id
Name
Level (if a monster)
Attack
Defense
Effect
Image file*

* When adding an image make sure to use the appropriate naming conventions for the file, which is:
Start of each word is a capital
No spaces in file name
